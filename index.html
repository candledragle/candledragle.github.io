Python 中from… import 与 import…. as的区别

from… import

​ 如 from A import b,相当于

​ import A

​ b = A.b

​ 再如：

​ “from t2 import var1” 相当于：

​ import t2

​ var1 = t2.var1

​ 在此过程中有一个隐含的赋值的过程

​ import… as

​ import A as B，给予A库一个B的别称，帮助记忆。
